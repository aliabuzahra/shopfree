# ShopFree Frontend

واجهة المستخدم الأمامية لمنصة ShopFree باستخدام React و Material-UI.

## المتطلبات

- Node.js 18+ 
- npm أو yarn

## التثبيت

```bash
npm install
```

## التشغيل

```bash
npm run dev
```

التطبيق سيعمل على `http://localhost:3000`

## البنية

- `src/pages/` - صفحات التطبيق
- `src/components/` - المكونات القابلة لإعادة الاستخدام
- `src/services/` - خدمات API
- `src/context/` - Context API لإدارة الحالة
- `src/utils/` - أدوات مساعدة
